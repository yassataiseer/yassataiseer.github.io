Python Crash Course Part 3

Hello everyone, welcome to the third Python Crash Course where we will be finishing off this series by learning about lists. Arguably the hardest part part of this crash course but with my help you can get through it.


Lists:

While coding we store data in variables however what happens if we need to store more than one piece of data in a variable. The solution for this is to use a data structure called Lists. Lists are a data structure found in every coding language that allows you to store an infinite number of data in one variable. Most languages only allow Lists of a certain type such as integers or strings. However thanks to Python's versatility you can use any data structure. Observe: 


a_list = ["STRING",100,True] # Make a list called “a_list”
print(a_list)
#Output: ['STRING', 100, True]

Inserting

As seen in the code above we can create a list and insert values when created but what if we want to add items into the list? We can simply do this via the .append() method 


a_list = []
a_list.append("HELLO")#.append method adds value to list
print(a_list)
#Output: ['HELLO']

We can also insert values at a specific index within the list using the .insert() like so:


a_list = []
a_list.append("HELLO")
print(a_list)
a_list.insert(0,"GOODBYE")#Insert GOODBYE at 0th index
print(a_list)
#Output: ['HELLO']
#output: ['GOODBYE', 'HELLO']

Deleting

Now let's suppose if we need to delete data from the list. We can do this via the .pop() method.


a_list = ["DATA1","DATA2","DATA3","DATA4"]
a_list.pop()
print(a_list)
#['DATA1', 'DATA2', 'DATA3']

Now let's suppose if we want to delete data from a list a specific index. To do so we will simply use the del statement. 


a_list = ["DATA1","DATA2","DATA3","DATA4"]
del a_list[1]
print(a_list)
# Output: ['DATA1', 'DATA3', 'DATA4']

The final thing we will explore is how to remove data from a list when specified by the user. To do this we can use the .remove() method. 


a_list = ["DATA1","DATA2","DATA3","DATA4"]
a_list.remove("DATA2")
print(a_list)
# Output: ['DATA1', 'DATA3', 'DATA4']
