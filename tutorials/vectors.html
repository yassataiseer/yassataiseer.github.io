Vectors In C++

Hey everyone today we will be exploring vectors, a very advanced data structure in C++ that is used in everyday use by programmers. They are able to hold data of any type and are dynamic often called lists or dynamic arrays as well.


Declarations:

In order to make a vector we need to import the given libraries below. No need to install them as they already exist as long as you have C++ installed on your machine. There are two main ways to create a vector one both ways below work 


vector new_vector = {1,2,3,4,5};//declare vector
vector vector_2(6,18);//Creates {18,18,18,18,18,18}
//Creating vector of size 6 and insert 18

Both ways lead to the same result and depending on your situation use one over the other


Indexing:

Indexing allows us to view a certain value within the vector; there are two ways to do this. 


cout << vector_2.at(1)<< endl;//Output: 2
cout << vector_2[1]<< endl; //Output: 2

The main difference between these two methods is that the first one will return errors if it is out of the index range. As a result, using .at(index) is better practise.


Front And Back:

We can view the first and last index of our vectors using the .front() and .back() method as follows: 


cout << new_vector.front() << endl;
cout << new_vector.back() << endl;

Sizing and Resizing:

When working with vectors we sometimes need to resize our vectors to be able to fit more data. We do this as so: 


new_vector.resize(100);//makes vector with size 100
cout << new_vector.size() << endl;//Output: 100

Pushback:

In order to insert more data into our vector, the data will be inserted into the end of the vector and sit on its very own index. This can be seen here. 


cout << new_vector.size() << endl;//output: 5
new_vector.push_back(7);//insert 7 into the end of this vector
cout << new_vector.size() << endl;//Output: 6

As seen above by adding another value to the vector the entire size of it changes!
