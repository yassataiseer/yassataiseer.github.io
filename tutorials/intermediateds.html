Intermediate Data Structures In C++

Hello everyone, today we will be learning about some intermediate data structures in the C++ programming language that are used in everyday programming life.


Structs:

Structs are a very complex data type(not a structure but important), it's more of a way to make your very own data types. Once created, it can be used to store multiple data types in one variable. These types can be accessed later on as well. Below we will grab the area of a triangle. 


 #include < iostream>   
using namespace std;   
struct triangle // making struct    
{     
  int base, height;     
    
};     
int main() {   
   struct triangle my_triangle;    //instance of struct
   my_triangle.base=8;    //added values to base and height
   my_triangle.height=5;   
   cout<<"Area of triangle is: "<<(my_triangle.base * my_triangle.height/2)<< endl;   
   // taken values and did mathematical operations
   return 0;   
}

Maps:

Maps are a data structure as also called a hashmap, that takes a key value and maps it to a specific value . In order to access a value you specify the key and accessing them are in O(1) notation and as a result move very quickly. If you ever are stuck in a coding problem try using a hashmap. Let's make a map that will store phone numbers(values) and the usernames(keys). 


 #include < iostream>   
#include < map>
 
using namespace std;   
 
int main() {   
   map< string, int> phone_numbers;
   //create map
   phone_numbers.insert(pair< string, int>("Yassa", 2132312));
   //insert into map
   cout << phone_numbers["Yassa"]<< endl;//print out map
   //output: 2132312
 
   return 0;

Stackss:

Stacks are very simple yet powerful that allow you to store data of any type. However you can only insert, delete, or see the top of the stack. Similar to a stack of plates. This concept is called LIFO(last in first out). Stacks are very fast and efficient with a complexity of O(1) A.K.A constant. Here is how to use one. 


 #include < iostream>
#include < stack>
using namespace std;
int main() {
   stack< int> stack;
   stack.push(1);
   stack.push(2);
   stack.push(3);
   stack.push(4);
   //Use .push(num) to insert num
   stack.pop();//remove num at top
   stack.pop();
 cout << stack.top()<< endl;//print top num
 }
