Part 1(Structs):

Last tutorial I only talked about structs briefly and did not talk about them in depth. As explained before structs are a data type that is used to hold data. Programmers can customize structs for their own benefits. To view a struct we can simply print it out but we can also view a value of a struct based on the reference given when the struct was created and given a value. 


 package main
 
import (
   "fmt"
)
type Coordinates struct{
   Latitude int
   Longitude int
}
func main() {
   x := Coordinates{5,10};
   fmt.Println(x.Latitude);//we can view struct values
   //prints out 5
}

The value of structs can be accessed after declaration and the value of each struct value can change after initial declaration as the values of a struct are not static however the size of the struct remains the same(e.g Two Ints ). We can change struct values like this: 


package main
 
import (
   "fmt"
)
type Coordinates struct{
   Latitude int
   Longitude int
}
func main() {
   x := Coordinates{5,10};
   x.Latitude = 22;
   fmt.Println(x);
   //output{22 10}
}

}

Part 2(Arrays):

The final thing we will learn in this series is Arrays. Arrays are the most basic data structure that are used everywhere. Arrays allow you to store a fixed amount of values of a certain data type(e.g Ints). Every value of an array is stored and the numerical location is called an “Index” and indexes start from 0 and onwards. The size value of an array cannot be changed otherwise there will be a huge error. 


 package main
 
 import (
    "fmt"
 )
  
 func main() {
    var a [2]string//initialize a string array
    //Array has a size of 2
    a[0] = "Hello"
    a[1] = "World"
    //Insert values into a
    fmt.Println(a[0], a[1]) //Output: Hello World
    fmt.Println(a)//Output: [Hello World]
    a[0]="Goodbye"
    //Change value of first index from "Hello" to "Goodbye"
    fmt.Println(a[0], a[1])//Output: Goodbye World
    fmt.Println(a)  //Output: [Goodbye World]
 }

We also do not need to specify every single value and can keep some empty/not specified they will simply be null values or empty 


package main
 
import (
   "fmt"
)
 
func main() {
   var a [2]string//initialize a string array
   //Array has a size of 2
   a[0] = "Hello"
   //Ive only given the value to the first index
   fmt.Println(a[0], a[1]) //Output: Hello
   fmt.Println(a)//Output: [Hello ]
   a[0]="Goodbye"
   //Change value of first index from "Hello" to "Goodbye"
   fmt.Println(a[0], a[1])//Output: Goodbye
   fmt.Println(a)  //Output: [Goodbye ]
}

We can also use the declarative operator(=:) to make arrays like this: 


package main
 
import (
   "fmt"
)
 
func main() {
   numbers := [6]int{2, 333, 5, 17, 151, 133}
   fmt.Println(numbers)//Outputs [2 333 5 17 151 133]
}
