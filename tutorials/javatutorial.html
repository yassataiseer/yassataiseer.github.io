Hello everyone, welcome to my Java crash course today I will be teaching the basics of Java at an introductory level. Java is an amazing programming language used in android applications, backend development, and system programs.


Variables:

The first thing we will be learning about variables. Variables are things that store data. There are different types of variables such as integers which are just numbers, doubles which are numbers with decimals, strings which are text(can include numbers), and booleans which are true or false statements. We can print these values out to be viewed into our terminal. 


 public class Main {
   public static void main(String[] args) {
       int number = 55;
       // declare Integer with value 55
       System.out.println(number);
       //prints it out
       //Output(55)
 
       double non_int = 55.50;
       // declare double called non_int
       non_int = non_int+number;
       //doubles and ints can be added as they are both numbers
       System.out.println(non_int);
       //prints it out(110.5)
      
       String text = "here is some text1234";//String
       System.out.println(text);
 
       Boolean check = true;//boolean
       System.out.println(check);
   }
 }

Variables Conversions:

We can also convert between different variables, this is so we can use them with other variables of the same type. For example adding strings together or numbers. Here are some ways to convert variables and utilize them.

Integer to String: 


 public class Main {
   public static void main(String[] args) {
       int age = 16;//my age as int
       String sentence = "my age is: ";
       String str_age = String.valueOf(age);//convert age into string
       System.out.println(sentence+str_age);
       // Output: my age is: 16
 
 }
}

String to Integer:


public class Main {
   public static void main(String[] args) {
 
       String age = "21";// my age as string
       int age_int = Integer.parseInt(age);
       System.out.println(age_int/7);
       //Output is my age in dog years(3)
 }
}

Loops:

Loops allow for code to be run over and over again for a certain amount of time. There are two types of loops, for-loops and while-loops.

While Loop:

While loops continuously run as long as long as a condition is met. 


public class Main {
   public static void main(String[] args) {
 
       int times = 0;
       while(times!=5){
           System.out.println("HELLO");
           times++;
       }//will run 5 times
       /*
       Output:
       HELLO
       HELLO
       HELLO
       HELLO
       HELLO
       */
 }
}

For loops:

For loops run for a certain period of time usually specified by a variable.


public class Main {
   public static void main(String[] args) {
 
       for(int times = 0;times< 5 ;times++){
           System.out.println("HELLO");
     
       } // will run 5 times
       /*
       Output:
       HELLO
       HELLO
       HELLO
       HELLO
       HELLO
       */
 }
}

As you can see both loops are very similar and as a result can accomplish the same things.


If-Else:

If statements are used to decide whether or not code should run or not. The code will run if the statement is true, you can have several if statements in one chunk by having if, then else-if, and else statements. 


 public class Main {
   public static void main(String[] args) {
 
       int age = 16;
       if(age>=18){
           System.out.println("You are an adult");
       }
       else{
           System.out.println("You are not an adult");
       }
       //Output: You are not an adult
 }
}

OR Statements:

We can also choose to run an if statement if one of many conditions are true. This is called an OR statement.For a better example: You will earn 5 dollars if either you mow the lawn OR take the trash out. If you do one or two of the tasks you will get 5 dollars. Here's a code example: 


public class Main {
   public static void main(String[] args) {
      
       boolean take_trash = true;//you took out the trash
       boolean mow_lawn = false;//you did not mow a lawn
       if(take_trash||mow_lawn){
           // || is OR
           System.out.println("you get 5 dollars");
       }
           // Since take_trash is true
           // this will run
           //Output: you get 5 dollars
 }
}

AND Statements

AND statements are used when we want all statements inside of the if statement to be true. For example you will earn 5 dollars if you take out the trash and you take out you will get 5 dollars, otherwise you will get nothing.Here's a code example: 


public class Main {
   public static void main(String[] args) {
      
       boolean take_trash = true;//you took out the trash
       boolean mow_lawn = false;//you did not mow a lawn
       if(take_trash&&mow_lawn){
        // && is AND
           System.out.println("you get 5 dollars");
       }
       else{
           System.out.println("you don't get 5 dollars");
       }
           // Since take_trash is true but mow_lawn is false
          // this won't run
          //Output: you don't get 5 dollars
 }
}
