Part 3(Playing sounds):

The next thing we will do is play our noise when clicked. We will also be printing out the key that is pressed in order for debugging. 


from pynput.keyboard import Key,Listener 
from pygame import mixer
def on_release(key):
    #we will create an exit button
    #so when the user clicks the 'esc' key
    #The program will exit

def on_press(key):
    print("{0} pressed".format(key)) # Prints of key
    mixer.init() #initialize mixer
    mixer.music.load('audio.mp3') #load audio file
    mixer.music.play() #play the file

with Listener(on_press=on_press, on_release=on_release) as listener:
    listener.join()

Part 4(Exiting the App):

In order to exit this app we can hit Ctrl+C however sometimes this will not work. To tackle this we will make a safe switch that will exit the program by simply clicking the Escape key.
