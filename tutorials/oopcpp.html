OOP For Begginers With C++

Hello everyone, today we will be learning about OOP(Object Oriented Programming) inside of C++. Before you start, make sure you have a basic understanding as to how C++ works. In this tutorial we will be building a grocery store class.


Making A Class:

In order to make a class you need to first of all think of a name of the class and type it as so: 


#main.cpp
#include < iostream>
#include < vector>
#include < string>
using namespace std;
class GroceryStore {       // The class
 
};
 
 
int main()
{
   GroceryStore();//Call class
   return 0;
}

Make sure the class is outside the main method or any other method. You will call the class simply as you would a function.


Constructors:

       }

Make sure to have your variables that you want to access/modify under the public: header as otherwise it will not be accessible as it would be private.


Methods:

Our classes would be useless had they not had any sort of tasks to do. In order to assign tasks we make methods that take constructor values and do something with them. Although they do need to take constructor values, they usually do. Let's make a method that checks to see if Bob can afford to buy his milk. 


#main.cpp
#include < iostream>
#include < vector>
#include < string>
using namespace std;
class GroceryStore {       // The class
 
   public:
       string item1;
       int item_price;
       int money;
       GroceryStore(string item,int price, int money_wallet){
           item1 = item;
           item_price = price;
           money = money_wallet;
       }
       bool Check(){//Method made
           if(item_price< money){
               cout << "The person can buy "<< item1 <<" as they are paying " << item_price;
               return false;
           }
           else{
               cout << "No they cannot buy "<< item1;
               return true;
           }
       }
 
};
 
 
int main()
{
   GroceryStore Bob("Milk",15,20);
   cout << Bob.Check() << endl;//Calling the check method and printing out values
   //Output: The person can buy Milk as they are paying 150
   return 0;
}

Make sure the class is outside the main method or any other method. You will call the class simply as you would a function.
