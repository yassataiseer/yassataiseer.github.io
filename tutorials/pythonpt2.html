Part 1(Variable Conversions):

Last time we explored several different data types like Integers and Strings. However, how can we use these data types hand in hand? Well the simple answer is you can't! Observe: 


 Integer = 12
text = "THIS IS TEXT "
print(text+Integer)
#Code will output an error

Now let's suppose if we want the Integer to add to the text like this THIS IS TEXT 12 in order to do so we will have to convert from an Integer data type to a string data type. We can convert any data type into any other data type as long as there is no part of the value that is specific to a certain data type. For example a string with letters cannot become an integer but if the string only has numbers it can convert. Observe: 


Integer = 12
text = "THIS IS TEXT "
Integer = str(Integer)
print(text+Integer)
# we have converted Integer into string
# we simply wrap the variable with str()
# Output: THIS IS TEXT 12
boolean = True
boolean = str(boolean)
print(boolean)
# we have converted Boolean into string
# we simply wrap the variable with str()
# Output: "True"
boolean = True
boolean = int(boolean)
print(boolean)
# we have converted Boolean into int
#Booleans are not based on text, they are binary

# we simply wrap the variable with int()

Part 2(Loops:):

Often when writing code we need it to repeat itself for a certain amount of times or until a certain condition is met. Python and every other coding language has a way of implementing this in a way called loops. Loops repeat code over and over for a certain amount of times. Python has two types of loops called For Loops and While Loops.

For Loops:

For loops are loops that run for a certain amount of time. When running they perform the same task over and over again. 


 for i in range(3):
   print("HELLO THERE")
#HELLO THERE
#HELLO THERE
#HELLO THERE

While Troop:

While loops keep running until a certain condition is met. They use boolean logic similar to If-Else statements. It is possible to write a while loop that runs forever to quit a while loop type Ctrl+C. An infinite while loop can be seen here:

   print("HELLO THERE")

number_run=3

Part 3(Functions):

Oftentimes when writing code we often repeat ourselves. This becomes a problem when writing huge amounts of code. Instead of constantly writing we can also use Functions which are containers with code that can be executed later. Functions return a value and that value can be used in our code. Python allows you to not return values but is horrible practise as every other language requires it. 


def function(): # use keyword "def" then function name
   number = 50*10
   return number
 
answer = function() #call the function
print(answer)
#Output: 500

Functions can have specific values called arguments that the user will specify themselves. This allows functions to be even more powerful as users only need to specify values and the code takes care of the rest. 


def function(number1): # use keyword "def" then function name
   number = number1*10
   return number
 
answer = function(200) #call the function with given value of 10
print(answer)
#Output: 2000

As seen here we specify an argument in our function that is required whenever calling our function.
