Golang Tutorial Part Two

Hello everyone, welcome to part two of our Golang tutorial today we will be looking at some more advanced concepts. Be sure to watch my previous video if you haven't seen it. In this video we will be learning about short declarations, structs and Maps. So let's dive right in!


Part 1(Short declarations ":=" ):

Short declarations in Golang are used to help speed up coding processes. Short delerations look like this ":=" Short declarations allow you to declare variables without specifying their type and the compiler will depict the type. You can also store functions inside short declarations as well as structs that we will get to later. We can also map more than one variable in one line. This is especially useful when using MySQL that we will discuss in a future tutorial. 


 package main
 
 import (
    "fmt"
    "reflect"
 )
 func main() {
    hiden_func := func() {
        fmt.Println("We are inside a function")
    }
    var age int = 5
    age1 := true // See we dont need to initialize what our variable type is!
    fmt.Println(age)
    fmt.Println(age1)
    fmt.Println(reflect.TypeOf(age1))
    hiden_func()
    a,b := 5,6
    fmt.Println(a,b)
 }

Part 2(Structs):

The next thing we will be exploring is called structs. Golang is unfortunately not an OOP language, in order to combat this structs were created and are basically watered down versions of classes. If you are coming from another NON-OOP language like C you will have no problem understanding this. Structs are a data type that allow developers to hold different values all together and can be used to store data in an organized fashion. 


 package main
 
import (
   "fmt"
)
type Coordinates struct {
   Latitude int
   Longitude int
}
func main() {
   coordinate := Coordinates{50,100};//We call upon the Coordinates struct like a class
   fmt.Println(coordinate);
}

Part 3(Maps):

In the final part of this tutorial we will be exploring Maps, one of the most advanced data structures to solve your problems. I have made a video on how Maps work in Java and Python. More information about Maps can be found there regardless if you want to learn about Java or Python. Maps allow you to store values with a certain key and are super fast. We will be making a Map and also using our Coordinates struct from the previous tutorial. 


 package main
 
 import (
    "fmt"
 )
 type Coordinates struct {
    Latitude int
    Longitude int
 }
 func main() {
    var data map[string]Coordinates//initialize a map
    data = make(map[string]Coordinates)//create an instance
    data["White House"]=Coordinates{20,11}//add key-values
    data["Red House"]=Coordinates{22,111}//add some more key-values
    fmt.Println(data["White House"])}//print out specific value based on key
