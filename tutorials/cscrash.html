C# Crash Course

Hello everyone, welcome to my crash course of the C# programming language. Everything I teach today will be on beginners difficulty and very similar to that of grade 11 comp sci.


Variables:

Variables are things that store data. There are different types of variables such as integers which are just numbers, doubles which are numbers with decimals, strings which are text(can include numbers), and booleans which are true or false statements. We can print these values out to be viewed into our terminal. 


 using System;
 
 namespace c_tutorial
 {
    class Program
    {
        static void Main(string[] args)
        {
            int number = 55;
            // declare Integer with value 55
            Console.WriteLine(number);
            //prints it out
            //Output(55)
  
            double non_int = 55.50;
            // declare double called non_int
            non_int = non_int+number;
            //doubles and ints can be added as they are both numbers
            Console.WriteLine(non_int);
            //prints it out(110.5)
  
            string text = "here is some text1234";//String
            Console.WriteLine(text);
  
            bool check = true;//boolean
            Console.WriteLine(check);
        }
    }
 }

Loops:

Loops allow for code to be run over and over again for a certain amount of time. There are two types of loops, for-loops and while-loops.

While Loop:

While loops continuously run as long as long as a condition is met. 


using System;
 
namespace c_tutorial
{
   class Program
   {
       static void Main(string[] args)
       {
           int times = 0;
           while(times!=5){
               Console.WriteLine("HELLO");
               times++;
           }//will run 5 times
           /*
           Output:
           HELLO
           HELLO
           HELLO
           HELLO
           HELLO
           */
       }
   }
}

For loops:

For loops run for a certain period of time usually specified by a variable.


using System;
 
namespace c_tutorial
{
   class Program
   {
       static void Main(string[] args)
       {
           for(int times = 0;times< 5 ;times++){
               Console.WriteLine("HELLO");
          
           } // will run 5 times
           /*
           Output:
           HELLO
           HELLO
           HELLO
           HELLO
           HELLO
           * /
       }
   }
}

As you can see both loops are very similar and as a result can accomplish the same things.


If-Else:

If statements are used to decide whether or not code should run or not. The code will run if the statement is true, you can have several if statements in one chunk by having if, then else-if, and else statements. 


 using System;
 
 namespace c_tutorial
 {
    class Program
    {
        static void Main(string[] args)
        {
            int age = 16;
            if(age>=18){
                Console.WriteLine("You are an adult");
            }
            else{
                Console.WriteLine("You are not an adult");
            }
            //Output: You are not an adult
        }
    }
 }

OR Statements:

We can also choose to run an if statement if one of many conditions are true. This is called an OR statement.For a better example: You will earn 5 dollars if either you mow the lawn OR take the trash out. If you do one or two of the tasks you will get 5 dollars. Here's a code example: 


using System;
 
namespace c_tutorial
{
   class Program
   {
       static void Main(string[] args)
       {
           bool take_trash = true;//you took out the trash
           bool mow_lawn = false;//you did not mow a lawn
           if(take_trash||mow_lawn){
               // || is OR
               Console.WriteLine("you get 5 dollars");
           }
           // Since take_trash is true
           // this will run
           //Output: you get 5 dollars
       }
   }
}

AND Statements

AND statements are used when we want all statements inside of the if statement to be true. For example you will earn 5 dollars if you take out the trash and you take out you will get 5 dollars, otherwise you will get nothing. 


using System;
 
namespace c_tutorial
{
   class Program
   {
       static void Main(string[] args)
       {
           bool take_trash = true;
           bool mow_lawn = false;
           if(take_trash&&mow_lawn){
            // && is AND
               Console.WriteLine("you get 5 dollars");
           }
           else{
               Console.WriteLine("you don't get 5 dollars");
           }
           // Since take_trash is true but mow_lawn is false
           // this won't run
           //Output: you don't get 5 dollars
       }
   }
}
