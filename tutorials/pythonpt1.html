Part 2(Data types):

The first thing we will be exploring is the different types of data understanding which are: Integers,Strings,Floats,and Booleans. These basic data structures are used in every program regardless of its complexity.

Booleans

Booleans are the most generic data types and are even found in hardware and are very similar to binary. With boolean's basis if-else statement logic is created which we will get into later. Booleans have two values True Or False this is especially useful when we need a basic answer such as is this car blue? We can simply use a boolean to answer this question. 


car_blue = False
print(car_blue)#prints out False

We put a # in front of any code we do not want the computer to read this is called Comments they make code more readable. We can change the value of a boolean just by changing what it is equal to like this: 


car_blue = False
print(car_blue)
#prints out False
car_blue = True
print(car_blue)
#prints out True

Strings

Strings are the only data type that can hold alphanumeric values and just like booleans can change in value as well as length. Programmers use strings to hold this such as sentences, usernames, passwords etc. 


car_blue = "The car is not blue"
print(car_blue)
#prints out:"The car is not blue"
car_blue = "The car is blue"
print(car_blue)
#prints out:"The car is  blue"

We can also add strings together to make bigger strings and ever multiply like this: 


car_blue = "The car is not blue"+" it is red"
print(car_blue)
#prints out:"The car is not blue it is red"
car_blue = "The car is blue"*3
print(car_blue)
#prints out:" The car is blue  The car is blue  The car is blue ""

Integers

Integers(A.K.A Ints) simply mean any whole number such as 1 or 400 however it does not include decimal numbers such as 1.15 or ever 5.0 even though 5.0 is a whole number it contains a decimal. Any decimal number is called.We can do all BEDMAS operations with the Int data type 


age = 15
print(age)
#prints out: 15
age+=10 
#adds 10 to current age
age = age*10
#multiplies age by 10
age = age/10
#divides age by 10
age = age^2
# age to the power of 2

Part 3(If-Else):

The next thing we will be looking at is If-Else statements. When coding sometimes we only want code to run when certain requirements are met otherwise it should not. Such as if someone has more than $100 in their bank account they can be allowed to withdraw $100. If-else statements are used here and if they did not exist the entire bank could go bankrupt 


 money = 5000
if money>100:#if statement is true will execute
    print("The person can withdraw $100")
else:
    print("The person cannot withdraw $100")
#output: "The person can withdraw $100"

As you can see the If-Else statements will only execute if a statement is True and as a result If-Else statements use boolean logic.

We can also use elif statements when we may want to check to see if another statement might be true as seen below:


money = 90
if money>100:#if statement is true will execute
    print("The person can withdraw $100")
elif money>50:
    print("Well you cant get $100 but you can get $50")
else:
    print("The person cannot withdraw $100")
#output: "Well you cant get $100 but you can get $50"

As the elif statement is true it will execute and will not look at any other boolean logic below it
